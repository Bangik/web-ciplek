## How to Install on Localy

Clone the project

```bash
  git clone https://github.com/Bangik/web-ciplek.git
```

Go to the project directory

```bash
  cd web-ciplek
```

Install dependencies

```bash
  composer install
```

Copy the .env file from .env.example

```bash
  cp .env.example .env
```

Edit .env file

APP_URL=http://localhost/...

DB_HOST=localhost

DB_NAME=...

DB_PORT=3306

DB_USER=...

DB_PASS=...
